THE BUZZ

SOCIAL MEDIA PLATFORM WITH SECURE BACKEND

OVERVIEW

THE BUZZ is a full-featured social media application built with Flutter and Supabase. The project demonstrates comprehensive software engineering practices, including secure backend architecture with REST APIs, Google OAuth integration, and robust SQL database management. Developed collaboratively in a 5-person team using agile methodologies, the project encompasses the complete lifecycle from UX design to deployment.

TECHNOLOGIES

  • FLUTTER
  • SUPABASE
  • REST API
  • SQL
  • JIRA
  • MAVEN
  • DOKKU

KEY FEATURES

  • SECURE BACKEND ARCHITECTURE — REST APIs with authentication and authorization protocols
  • GOOGLE OAUTH INTEGRATION — Seamless third-party authentication
  • SQL DATABASE MANAGEMENT — Robust relational database design for user data and social connections
  • CROSS-PLATFORM DEVELOPMENT — Flutter implementation for iOS and Android
  • UNIT TESTING — Comprehensive test coverage for reliability
  • CONTINUOUS DEPLOYMENT — Dokku deployment pipeline for production releases

DEVELOPMENT PROCESS

This project was managed using Jira and Lucid, employing agile sprint methodology. The team utilized Maven for build automation and dependency management. We practiced continuous integration and deployment using Dokku, ensuring smooth releases and reliable production environments. This collaborative experience strengthened my ability to work in teams, manage complex technical requirements, and deliver high-quality software products.
